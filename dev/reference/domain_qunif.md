# Transform a Numeric Value to a Sample

Return a valid sample from the given
[`Domain`](https://paradox.mlr-org.com/dev/reference/Domain.md), given a
value from the interval `[0, 1]`.

## Usage

``` r
domain_qunif(param, x)
```

## Arguments

- param:

  (`Domain`).

- x:

  `numeric` between 0 and 1.

## Value

`any` – format depending on the `Domain`.
