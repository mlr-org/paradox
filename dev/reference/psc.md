# Create a ParamSet Collection

Creates a
[`ParamSetCollection`](https://paradox.mlr-org.com/dev/reference/ParamSetCollection.md).

## Usage

``` r
psc(...)
```

## Arguments

- ...:

  (any)  
  The
  [`ParamSet`](https://paradox.mlr-org.com/dev/reference/ParamSet.md)s
  from which to create the collection.
