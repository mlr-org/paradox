# Whether a Given Domain is Categorical

This should generally be `TRUE` for categorical
[`Domain`](https://paradox.mlr-org.com/dev/reference/Domain.md)s, such
as [`p_fct()`](https://paradox.mlr-org.com/dev/reference/Domain.md) or
[`p_lgl()`](https://paradox.mlr-org.com/dev/reference/Domain.md), and
`FALSE` otherwise.

## Usage

``` r
domain_is_categ(param)
```

## Arguments

- param:

  (`Domain`).

## Value

`logical`.
