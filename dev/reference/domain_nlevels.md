# The Number of Levels of a Given Domain

This should be the number of discrete possible levels for discrete type
[`Domain`](https://paradox.mlr-org.com/dev/reference/Domain.md)s such as
[`p_int()`](https://paradox.mlr-org.com/dev/reference/Domain.md) or
[`p_fct()`](https://paradox.mlr-org.com/dev/reference/Domain.md), and
`Inf` for continuous or untyped parameters.

## Usage

``` r
domain_nlevels(param)
```

## Arguments

- param:

  (`Domain`).

## Value

`numeric`.
